## Добавление новой возможности

#### 25729
- Добавлена возможность автоматически завершать работу программы на мобильном терминале при привышении времени простоя;
- Добавлена возможность автоматически закрывать сессию после завершения программы на 
моб. терминале при привышении времени простоя;

- Завершать терминальную сессию при достижении времени простоя


#### 25732
В соответствии с __Приказом ФНС России от 19.12.2023 № ЕД-7-26/970__, в БУХте внесены изменения в режим выгрузки документов в xml-файл в формате УПД (форма __Электронные документы. Экспорт Документа в XML__)

Т.к. с 1.01.2026 необходимо при наличии аванса заполнять в строке 5б данные по авансовому счету-фактуре.
- Если документ на аванс один, то для корректной выгрузки номера и даты документа на аванс, в запросе по документу следует добавить поля: __СопрДокФХЖ_РеквНомерДок__ - тип строка - номер документа на аванс, __СопрДокФХЖ_РеквДатаДок__ - тип дата - дата авансового документа.
- Если документов на аванс несколько, то данные о них выгружаются по новому запросу __Электр.документ. Экспорт в  XML Документ Аванс__ (поля запроса называются так же, как новые поля в запросе по документу - __СопрДокФХЖ_РеквНомерДок и СопрДокФХЖ_РеквДатаДок__).
- Если такой запрос есть, то данные берутся из него, если запроса нет или в результате запроса нет строк с заполненным номером документа - данные берутся из соответствующих полей запроса по документу.
- Также добавлена возможность выгружать Должность и ФИО руководителя в строке 12 УПД, для этого в запросе по ДОКУМЕНТУ должны быть строковые поля: __РабОргПрод_Должность, РабОргПрод_Фамилия, РабОргПрод_Имя, РабОргПрод_Отчество__.

В соответствии с __приказом ФНС России от 01.09.25 № ЕД-7-11/757@__, в справочники БУХты добавлены новые коды налогоплательщика, действующие с 2026 г:
- «0» — налогоплательщик, имеющий статус иностранного агента;
- «9» — налогоплательщик — налоговый резидент и гражданин государства — члена Евразийского экономического союза, получающий доходы от осуществления трудовой деятельности на территории Российской Федерации, не является налоговым резидентом Российской Федерации.
Эти коды следует указать в Личной карточке сотрудника в таблице __ЗП Налоги__ - на закладке __Налоги__ в карточке на закладке __ИМНС\ПФР__ (для корректного расчета НДФЛ на закладке __Налоги__ в поле __Удерживать подоходный__ следует указать __Резидент__ для кода налогоплательщика 9 и __Неризидент__ для кода 0).

#### 25743
Доработан алгоритм сбора данных для выгрузки отчета 6НДФЛ (форма __З/П. Экспорт в XML-файл для ГНИ 2025__)

#### 25747
Доработан алгоритм заполнения даты выплаты в ежемесячных шифрах расчетного листа З/П при расчете ЗП, теперь при определении даты выплаты ежемесячных шифров учитывается, не превышает ли рассчитанная дата выплаты последний день месяца и если превышает, дата выплаты приравнивается к последнему дню месяца.

## Информация для разработчиков

#### 25726
Добавлен класс __TbmPhoneValidator__ и метод __TestPhone(APhoneNumber: string)__.
Возвращает объект __TbmPhoneNumber__.

```delphi
  TbmPhoneNumber = class
  public
    property CountryCode: Integer
    property NationalNumber: string
    property RegionId: string
    property IsValid: Boolean
    property IsMobile: Boolean
    property FormattedNumber: string
    property ExtendedText: string
  end;
```

```basic
  if TelEdit.Modified and (TelEdit.Value<>"") then
  'получаем экземпляр TbmPhoneNumber
    pPhoneNumber=TbmPhoneValidator.TestPhone(TelEdit.Value)

    if pPhoneNumber.RegionID="" then
      SaveAllow=False
      SaveAllowMsg="Не распознан номер региона"
      pPhoneNumber.Free
      exit sub
    end if

    if pPhoneNumber.NationalNumber="" then
      SaveAllow=False
      SaveAllowMsg="Не распознан тел. номер"
      pPhoneNumber.Free
      exit sub
    end if

    bmInformation("Номер распознан как: "+pPhoneNumber.ExtendedText)
    'Проверка корректности номера, если регион и тело номера есть
    if pPhoneNumber.IsValid=false then
      SaveAllow=False
      SaveAllowMsg="Не верный формат номера"
    else
      SaveAllow=True
    end if
    pPhoneNumber.Free
  end if
```
