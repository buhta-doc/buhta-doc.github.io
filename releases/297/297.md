## Добавление новой возможности

#### 26807
Добавлена возможность выгружать форму стат.наблюдения П-6 в новом формате (форма __Стандартная. Архивный отчет в эл.виде (Стат.формы)__).

#### 26814
Доработано заполнение шаблона __Стандартный. Налоговая декларация по НДС 2026__ по запросу по sql-алгоритмам:
- строка 070 раздела 3.1 теперь заполняется и для ставки НДС 22% (форма __Стандартная. Декларация по НДС 2015. Экспорт в XML__).
- Для корректного заполнения строки 070 в запросе по SQL-алгоритмам НЕ ДОЛЖНО БЫТЬ ПОЛЯ с названием __[Раздел 3.1 строка 070]__, ДОЛЖНО быть поле с названием __[Раздел 3.1 строка 070_22проц]__.


## Исправление ошибок

#### 26811
Исправлена ошшибка импорта договора из xml-файла в формате __УПД__ (форма __Электронные документы. Импорт из XML__).

#### 26815
Исправлена ошибка при выгрузке отчета __ЕФС-1__ (форма __З/П Форма ЕФС-1__). Не выгружались корректировочные сведения зв пред.годы в Разделе 1.2 (исправлен запрос __З/П ЕФС1 Печать Раздел 1__).


## Информация для разработчиков

1. Теперь значение в поле __LastExit__ таблицы __UserName__ обновляется при отключении пользователя по следующим причинам:
- Превышен интервал времени простоя; 
- Сессия пользователя отключена из окна __Кто работает в сети__;

2. Для временных таблиц, при открытии карточки редактирования, не выставляется блокировка на текущую запись.
